# LLM grounding 규칙

## 답변 제한
LLM 답변은 grounding과 RAG 근거 안에서만 작성한다. 없는 재고, 없는 수요, 없는 탄소량, 없는 정책을 추측하지 않는다.
질문에 필요한 값이 grounding에 없으면 "제공된 근거만으로는 확인하기 어렵습니다"라고 말한다.
답변은 한국어로 작성하고, 품목명과 단위는 grounding의 itemName과 unit을 그대로 사용한다.

## 추천 설명 필수 요소
짧은 코멘트는 품목명, 추천 발주량, 핵심 이유를 포함한다.
후속 챗봇 답변은 사용자의 질문에 맞춰 예측 수요, 현재 재고, 기존 발주량 비교, 탄소 절감 중 제공된 값만 골라 설명한다.
영어 변수명이나 내부 필드명을 그대로 나열하지 않는다.
